# Sample Hardhat Project

This is a completely decentralized Nft marketplace, which allows to sell your ERC721 token


# Build and Test
Use node version 14 or 16
```
npm install
```

# Deployemnent
```
npx hardhat deploy --network <network>
```

# Test
```
npx hardhat test
```
